# Kuya's Streetfood POS System
 
A simple Point of Sale (POS) system built for a Filipino street food restaurant. This was made as a final exam project to demonstrate JavaScript DOM manipulation and fundamental JavaScript concepts.
 
---
 
## Features
 
- **Menu Display** — Shows food items with images, names, and prices in a card grid layout
- **Add to Order** — Enter a quantity and click "Add to order" to add items to the order panel
- **Search Bar** — Filter menu items by name in real time
- **Category Filter** — Switch between All, Streetfood, Burgers, and Drinks
- **Order Summary** — Right panel shows all added items with quantities and subtotals
- **Remove Items** — Click the ✕ button to remove an item from the order
- **Clear Order** — Clears all items from the current order
- **Live Change Calculator** — Automatically shows change (or shortage) as you type the payment amount
- **Pay Button** — Processes the payment and shows a receipt
- **Print Receipt** — Opens a printable receipt in a new window
---
 
## How to Use
 
1. Open `index.html` in any web browser
2. Browse the menu or use the search bar / category buttons to find items
3. Set the quantity and click **Add to order**
4. Once done ordering, type the payment amount in the input field
5. Click **Pay!** to complete the transaction and view the receipt
